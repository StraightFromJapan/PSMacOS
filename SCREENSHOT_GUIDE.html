üì∏ Screenshots to Capture

Screenshot 1: Main Studio View

Save as: 01-main-studio.png

What to capture:

  • The main view with sample pads numbered 1-100
  • Make sure the grid of sample buttons is visible
  • Include the top menu bar and any side controls

Steps:

  1. Open Psychological Studio
  2. Press Win + Shift + S
  3. Select the entire main window
  4. Paste into Paint (Ctrl+V)
  5. Save as: docs\screenshots\01-main-studio.png

Screenshot 2: Arrangement View

Save as: 02-arrangement-view.png

What to capture:

  • Click "Arrangement View" button to open the timeline
  • Add a few clips to the timeline so it's not empty
  • Show multiple tracks with clips
  • Include the playback controls (Play, Stop, BPM)

Screenshot 3: Effects Popup - Basic Tab

Save as: 03-effects-basic.png

What to capture:

  • In Arrangement View, right-click any clip
  • Select "Effects" from the menu
  • Make sure the Basic tab is selected
  • Capture the popup showing Volume, Speed, Delay, Reverb, Filter controls

Screenshot 4: Effects Popup - EQ Tab

Save as: 04-effects-eq.png

What to capture:

  • In the same Effects popup, click the EQ tab
  • Show the 8-band equalizer with visual canvas
  • Include all frequency band controls

Screenshot 5: Effects Popup - LFO Tab

Save as: 05-effects-lfo.png

What to capture:

  • Click the LFO tab in Effects popup
  • Show all 4 LFO controls
  • Include the waveform, rate, and depth settings

Screenshot 6: Effects Popup - Automation Tab

Save as: 06-effects-automation.png

What to capture:

  • Click the Automation tab in Effects popup
  • Show all 4 automation lanes
  • Include the curve controls and canvas

Screenshot 7: Control Panel

Save as: 07-controls.png

What to capture:

  • Close the Effects popup
  • Focus on the top control bar
  • Show Play, Stop buttons, BPM slider
  • Include File menu and any transport controls
  • Can be a zoomed crop of just the top control area

Screenshot 8: Clip Context Menu

Save as: 08-clip-context-menu.png

What to capture:

  • Right-click on a clip in Arrangement View
  • Capture the context menu showing:
    • Effects option
    • Edit option
    • Delete option
  • Include part of the clip and timeline for context
